# Study

Build projects that show:

- Repos and branch policies
- Boards and work item tracking
- YAML CI/CD
- Azure deployment
- Secrets management
- Docker and ACR
- AKS or App Service deployment
- Terraform or Bicep
- Monitoring and rollback

Mini task:
