# Git Workflow

Common flow:

```txt
Create branch -> Commit changes -> Push branch -> Open PR -> Review -> Merge -> Pipeline runs
```

Useful commands:

```bash
git status
git branch
git checkout -b feature/login
git add .
git commit -m "add login"
git push origin feature/login
```

Good practice:

- Small commits
- Clear branch names
- Pull request reviews
- Protected main branch
