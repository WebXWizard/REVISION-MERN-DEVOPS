# Cheatsheet

Azure DevOps:

```txt
Platform for planning, code, CI/CD, packages, testing, and delivery.
```

Repos:

```txt
Git repositories with PRs and branch policies.
```

Boards:

```txt
Work tracking using work items, backlogs, and sprints.
```

Pipelines:

```txt
Build, test, and deploy automation.
```

Artifacts:

```txt
Package feeds for NuGet, npm, Maven, Python, and Universal Packages.
```

Service connection:

```txt
Secure connection from pipeline to Azure or external services.
```
