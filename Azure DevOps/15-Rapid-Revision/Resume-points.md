# Resume Points

Example Azure DevOps resume bullets:

- Built YAML CI/CD pipelines with build, test, artifact publishing, deployment, approval, and rollback stages.
- Configured Azure Repos branch policies with PR reviews and build validation to protect the main branch.
- Deployed applications to Azure App Service using Azure Pipelines, environment approvals, and secure variables.
- Built Docker images in Azure Pipelines and pushed immutable tags to Azure Container Registry.
- Deployed containerized applications to AKS using Kubernetes manifests or Helm.
- Integrated Azure Key Vault with pipelines to manage secrets securely.
- Provisioned Azure infrastructure using Terraform or Bicep through reviewed pipeline stages.
- Added monitoring with Azure Monitor, Application Insights, and Log Analytics for deployed applications.
