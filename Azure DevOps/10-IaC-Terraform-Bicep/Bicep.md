# Bicep

Bicep is Azure's domain-specific language for deploying Azure resources.

Benefits:

- Cleaner than ARM JSON
- Azure native
- Good tooling
- Works with Azure Resource Manager

Use Bicep for:

- Azure-native infrastructure
- Repeatable templates
- Modular resource deployments

Terraform vs Bicep:

```txt
Terraform is multi-cloud. Bicep is Azure native.
```
