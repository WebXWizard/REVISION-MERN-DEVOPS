# 12 Week Azure DevOps Plan

Week 1:

- Azure fundamentals, subscriptions, resource groups, regions, identity, and RBAC.

Week 2:

- Azure DevOps organization, projects, services, permissions, and basic workflow.

Week 3:

- Git, Azure Repos, branching, pull requests, branch policies, and code reviews.

Week 4:

- Azure Boards, work items, backlogs, sprints, dashboards, and agile workflow.

Week 5:

- YAML pipeline basics, triggers, variables, stages, jobs, tasks, and build artifacts.

Week 6:

- CI/CD release flow, environments, approvals, gates, deployment jobs, and rollback.

Week 7:

- Agents, agent pools, self-hosted runners, service connections, permissions, and secrets.

Week 8:

- Azure Artifacts, feeds, NuGet, npm, package versioning, and dependency management.

Week 9:

- Docker, Azure Container Registry, image tagging, and container deployment pipeline.

Week 10:

- AKS, Kubernetes manifests, Helm, ingress, secrets, ConfigMaps, and deployment debugging.

Week 11:

- Terraform, Bicep, ARM templates, Azure Monitor, Key Vault, Policy, and governance.

Week 12:

- Build projects, revise top questions, and practice mock interviews.
