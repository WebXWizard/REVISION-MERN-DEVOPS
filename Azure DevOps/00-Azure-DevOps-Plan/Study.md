# Azure DevOps Study Plan

Azure DevOps is a Microsoft platform for planning, coding, building, testing, packaging, and deploying software.

Focus areas:

- Azure DevOps services
- Git and branch policies
- Boards and agile planning
- YAML pipelines
- Build and release automation
- Agents and service connections
- Azure deployments
- Artifacts and package feeds
- Docker, ACR, AKS, and Kubernetes
- IaC with Terraform and Bicep
- Monitoring, security, and governance

Suggested order:

1. Azure and Azure DevOps fundamentals
2. Git and Azure Repos
3. Boards and agile workflow
4. YAML CI pipelines
5. CD releases and environments
6. Agents and service connections
7. Artifacts and packages
8. Docker and ACR
9. AKS and Kubernetes deployment
10. Terraform and Bicep
11. Monitoring, security, and governance
12. Projects and mock interviews
