# Troubleshooting Round

Scenario:

```txt
The production deployment failed after a successful build.
```

Answer steps:

1. Check failed stage and task logs.
2. Confirm service connection permission.
3. Check environment approval or gates.
4. Verify artifact exists.
5. Check target Azure resource health.
6. Compare recent YAML or variable changes.
7. Rollback if production is affected.
8. Document root cause and prevention.
