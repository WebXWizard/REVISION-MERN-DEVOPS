# Resume Points

Example Automation Architect resume bullets:

- Designed automation workflows for user onboarding, cloud provisioning, release promotion, reporting, and incident remediation.
- Created automation assessment criteria based on frequency, rule clarity, business value, risk, and maintenance cost.
- Built API-based automation patterns with retries, idempotency, error handling, audit logs, and notifications.
- Designed Infrastructure as Code automation using Terraform modules, review gates, approvals, and remote state concepts.
- Defined governance controls for automation including least privilege, secret management, approvals, audit trails, and ownership.
- Created monitoring plans for automation success rate, duration, failures, retries, queue time, and business outcomes.
