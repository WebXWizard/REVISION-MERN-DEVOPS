# Last Minute Notes

Remember:

- Start with business problem, not tool.
- Automation should be measurable.
- Not every process should be automated.
- Prefer APIs and events over UI automation when possible.
- Handle failures, retries, duplicates, and partial completion.
- Add audit logs for important actions.
- Secrets must be stored securely.
- Risky actions need approvals.
- Automation needs ownership and maintenance.
- Strong answers include ROI and governance.
