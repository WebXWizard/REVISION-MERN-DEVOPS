# Commands

Python:

```bash
python --version
python script.py
pip install requests
```

PowerShell:

```powershell
Get-Process
Invoke-RestMethod -Uri "https://api.example.com"
```

Bash:

```bash
curl -s https://api.example.com
jq .
cron
```

Terraform:

```bash
terraform init
terraform plan
terraform apply
```
