# Cheatsheet

Automation Architecture:

```txt
Designing reliable, secure, observable automation systems that reduce manual work and business risk.
```

Good candidate:

```txt
Frequent, rule-based, measurable, stable, and high-value.
```

Workflow orchestration:

```txt
Coordinating tasks, dependencies, approvals, retries, and state.
```

Idempotency:

```txt
Running the same action multiple times produces the same safe result.
```

ROI:

```txt
Time saved and risk reduced compared with build and maintenance cost.
```
