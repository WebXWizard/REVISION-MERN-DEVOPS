# Automation Architect Answer Framework

Use this format for interview answers:

```txt
Manual process -> Pain and risk -> Automation candidate score -> Architecture -> Integrations -> Controls -> Monitoring -> ROI -> Improvement
```

Example:

```txt
For manual user onboarding, I would map the workflow, identify systems involved, define approvals, automate account creation through APIs, store secrets securely, log every action, alert on failures, and measure time saved and error reduction.
```

Good answers include:

- Process understanding
- Automation fit
- Architecture
- Integration points
- Security controls
- Exception handling
- Monitoring
- Business value
