# Last Minute Notes

Remember:

- Always connect security to risk.
- Mention assets, threats, controls, detection, and response.
- Least privilege is a default answer for access.
- Backend must enforce authorization.
- Secrets should not be in Git, images, or logs.
- Patching should be prioritized by exposure and impact.
- Good alerts are actionable.
- Incident response needs evidence and timeline.
- Compliance needs proof, not only claims.
- Strong answers include prevention and detection.
