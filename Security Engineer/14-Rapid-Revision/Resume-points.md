# Resume Points

Example Security Engineer resume bullets:

- Created threat models for web applications and mapped risks to preventive and detective controls.
- Hardened Linux servers by reviewing SSH, firewall rules, users, patches, services, and logs.
- Implemented DevSecOps checks including secret scanning, dependency scanning, SAST, IaC scanning, and container image scanning.
- Designed secure cloud architecture using least privilege IAM, private networking, audit logging, and secure storage configuration.
- Built incident response runbooks for suspicious login, leaked secret, and high-risk vulnerability scenarios.
- Practiced Kubernetes security using RBAC, network policies, resource limits, and admission policy controls.
