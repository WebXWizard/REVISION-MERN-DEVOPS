# Cheatsheet

CIA:

```txt
Confidentiality, Integrity, Availability.
```

Risk:

```txt
Likelihood multiplied by impact.
```

Least privilege:

```txt
Grant only permissions required for the job.
```

Defense in depth:

```txt
Multiple layers of security controls.
```

Incident response:

```txt
Detect, contain, eradicate, recover, and learn.
```
