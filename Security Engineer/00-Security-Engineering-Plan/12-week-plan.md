# 12 Week Security Engineer Plan

Week 1:

- Security fundamentals, CIA triad, risk, threats, controls, and defense in depth.

Week 2:

- Networking, firewalls, segmentation, DNS, TLS, VPN, IDS, IPS, and WAF.

Week 3:

- Linux hardening, endpoint security, logs, patching, and secure access.

Week 4:

- Cloud IAM, least privilege, security groups, private networks, audit logs, and secure architecture.

Week 5:

- Web application security, OWASP Top 10, API security, input validation, and auth security.

Week 6:

- DevSecOps, SAST, DAST, SCA, secret scanning, image scanning, and CI/CD security.

Week 7:

- SOC basics, SIEM, log analysis, alert triage, incident response, and threat hunting.

Week 8:

- Vulnerability management, CVSS, prioritization, remediation, and risk acceptance.

Week 9:

- Secrets management, encryption, hashing, PKI, certificates, and key rotation.

Week 10:

- Container and Kubernetes security, RBAC, network policies, admission policies, and runtime security.

Week 11:

- Compliance, GRC, audit evidence, policies, standards, and risk registers.

Week 12:

- Build projects, revise top questions, and practice mock interviews.
