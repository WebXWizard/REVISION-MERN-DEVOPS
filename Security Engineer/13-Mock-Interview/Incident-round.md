# Incident Round

Scenario:

```txt
An admin account logged in from an unusual country and changed IAM permissions.
```

Answer steps:

1. Confirm alert and account.
2. Preserve logs.
3. Disable or restrict account.
4. Revoke sessions and rotate credentials.
5. Review changes made.
6. Restore permissions.
7. Investigate root cause.
8. Improve detection and MFA controls.
