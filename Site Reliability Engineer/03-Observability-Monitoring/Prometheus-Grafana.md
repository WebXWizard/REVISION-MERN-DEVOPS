# Prometheus And Grafana

Prometheus:

- Scrapes metrics from targets.
- Stores time-series data.
- Uses PromQL for queries.
- Sends alerts through Alertmanager.

Grafana:

- Visualizes metrics.
- Builds dashboards.
- Helps compare service behavior over time.

Common SRE dashboards:

- API latency
- Error rate
- Request rate
- CPU and memory
- Disk and network
- Kubernetes pod health
- Database connections
