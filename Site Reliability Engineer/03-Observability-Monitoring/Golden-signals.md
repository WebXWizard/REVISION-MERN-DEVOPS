# Golden Signals

Latency:

- How long requests take.
- Track p50, p95, and p99.

Traffic:

- How much demand the service receives.
- Track requests per second.

Errors:

- Failed requests or incorrect responses.
- Track error rate by route and status code.

Saturation:

- How full the system is.
- Track CPU, memory, disk, queue depth, and connection pools.

I alert on symptoms users feel, then use dashboards and logs to find the cause.
