# Metrics, Logs, And Traces

Metrics are numeric time-series data.

Examples:

- Request count
- Error rate
- CPU usage
- Memory usage
- p95 latency

Logs are event records.

Examples:

- Error stack trace
- Request failure
- Deployment event

Traces show request flow across services.

Use traces for:

- Microservice latency
- Dependency bottlenecks
- Slow database calls
