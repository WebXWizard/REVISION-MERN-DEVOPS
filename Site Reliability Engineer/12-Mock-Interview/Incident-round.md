# Incident Round

Practice scenario:

```txt
The API error rate jumped from 1 percent to 20 percent after deployment.
```

Answer steps:

1. Confirm user impact.
2. Check dashboards and alerts.
3. Compare with recent deployments.
4. Rollback if new release is likely cause.
5. Monitor recovery.
6. Investigate logs and traces.
7. Write postmortem and follow-ups.
