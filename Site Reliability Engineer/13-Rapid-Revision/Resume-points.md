# Resume Points

Example SRE resume bullets:

- Built Prometheus and Grafana dashboards for API latency, error rate, traffic, and resource saturation.
- Created runbooks for high latency, high error rate, disk usage, and Kubernetes pod failures.
- Implemented health-check automation with logging, retry limits, and safe service restart flow.
- Practiced Kubernetes troubleshooting using logs, events, probes, rollout status, and rollback.
- Defined sample SLIs and SLOs for API availability and latency with error budget tracking.
- Designed incident response workflow with severity levels, communication updates, and postmortem action items.
