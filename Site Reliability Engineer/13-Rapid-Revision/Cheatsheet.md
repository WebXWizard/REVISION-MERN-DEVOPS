# Cheatsheet

SRE:

```txt
Software engineering applied to operations and reliability problems.
```

SLI:

```txt
Measurement of service behavior.
```

SLO:

```txt
Target reliability level.
```

SLA:

```txt
External promise, often business or legal.
```

Error budget:

```txt
Allowed unreliability before reliability work becomes priority.
```

Golden signals:

```txt
Latency, traffic, errors, saturation.
```
