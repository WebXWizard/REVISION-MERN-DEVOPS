# Last Minute Notes

Remember:

- Always mention user impact.
- Use metrics, logs, and traces as evidence.
- Mitigate first, investigate deeper after stability.
- Rollback is valid if deployment caused the issue.
- Postmortems should be blameless.
- Alerts should be actionable.
- SLOs should match user journeys.
- Automation should have safety limits.
- Backups must be restore-tested.
- Strong answers include prevention.
