# Project 3: Auto-remediation

Goal:

```txt
Build a safe script that detects a known issue and performs a controlled fix.
```

Example:

- Check service health.
- Confirm failure with multiple checks.
- Restart service.
- Verify recovery.
- Send notification.
- Log all actions.

Safety features:

- Dry-run mode
- Retry limits
- Clear logs
- Manual escalation if restart fails
