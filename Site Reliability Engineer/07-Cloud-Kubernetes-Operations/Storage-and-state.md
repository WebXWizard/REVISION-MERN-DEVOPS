# Storage And State

Kubernetes storage concepts:

- PersistentVolume
- PersistentVolumeClaim
- StorageClass
- StatefulSet

SRE concerns:

- Backup and restore
- Data corruption
- Disk full
- Slow I/O
- Node failure
- Stateful workload migration

Good habit:

```txt
Never assume a backup works until restore has been tested.
```
