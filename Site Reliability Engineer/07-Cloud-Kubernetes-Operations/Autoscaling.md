# Autoscaling

Common autoscaling types:

- Horizontal Pod Autoscaler: scales pods.
- Cluster Autoscaler: scales nodes.
- Vertical Pod Autoscaler: adjusts pod resources.

Signals:

- CPU
- Memory
- Request rate
- Queue depth
- Custom metrics

Risks:

- Scaling too slowly
- Scaling too aggressively
- Missing resource requests
- Dependency bottlenecks
