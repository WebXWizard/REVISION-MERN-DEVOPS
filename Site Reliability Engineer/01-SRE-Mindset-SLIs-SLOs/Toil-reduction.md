# Toil Reduction

Toil is manual, repetitive work that does not create lasting value.

Examples:

- Restarting the same service manually every day
- Copying logs by hand
- Manually checking disk usage
- Repeated deployment steps
- Manual user access changes

Good automation candidates:

- Clear steps
- Frequent repetition
- Low decision complexity
- Easy rollback
- Measurable output

SRE goal:

```txt
Keep operational work sustainable by automating repeatable tasks and improving systems.
```
