# Study

The SRE mindset is about balancing reliability and feature delivery.

Learn these ideas deeply:

- Reliability is measured from the user's point of view.
- Every system has a reliability target.
- Error budgets help decide when to ship and when to stabilize.
- Toil should be reduced through automation.
- Postmortems should be blameless and action-oriented.

Mini task:

- Pick one app and define one availability SLO and one latency SLO.
