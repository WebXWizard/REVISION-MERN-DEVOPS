# SRE vs DevOps

DevOps is a culture and practice that improves collaboration between development and operations.

SRE is a specific engineering discipline that uses software practices to solve operations and reliability problems.

Simple comparison:

```txt
DevOps: improve delivery and collaboration
SRE: measure and protect reliability
```

SRE responsibilities:

- Define reliability goals.
- Build monitoring and alerting.
- Automate repeated operations.
- Handle incidents.
- Improve systems after failures.
- Reduce risk in deployments.
