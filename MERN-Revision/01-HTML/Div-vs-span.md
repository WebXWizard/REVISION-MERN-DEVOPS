# Div vs Span

`div` block-level container hai. `span` inline container hai.

Why used:

- `div`: layout grouping
- `span`: small inline text styling

Example:

```html
<div class="card">
  <p>Hello <span class="highlight">MERN</span> developer</p>
</div>
```

Common mistakes:

- Inline text ke liye unnecessary `div`.
- Layout ke liye too many meaningless wrappers.

Interview questions:

- Difference between div and span?
- Are div and span semantic?

Mini task:

- Create a paragraph with one highlighted word using `span`.
