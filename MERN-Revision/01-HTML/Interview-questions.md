# HTML Interview Questions

Concept questions:

1. What is HTML?
2. What is semantic HTML?
3. Difference between `div` and `span`.
4. Difference between inline and block elements.
5. What are meta tags?
6. Why is `alt` important in images?
7. What is the use of `label` in forms?
8. Difference between `localStorage` and `sessionStorage`.
9. Cookies vs sessions.
10. What is accessibility?

Coding questions:

1. Create a semantic page layout.
2. Create a login form with email and password.
3. Create a table for student records.
4. Add video with multiple source formats.

Scenario questions:

1. A form is not accessible for screen readers. What will you improve?
2. Website is not ranking on Google. Which HTML basics will you check?
3. User closes tab and data should be removed. Which storage will you use?

Follow-up questions:

1. Can semantic HTML improve SEO?
2. Should JWT be stored in localStorage? Why or why not?
3. When would you use cookies?
