# Semantic Tags

Semantic tags HTML elements ko meaning dete hain.

Examples:

- `header`
- `nav`
- `main`
- `section`
- `article`
- `aside`
- `footer`

Why used:

- SEO improve hota hai.
- Accessibility better hoti hai.
- Code readable hota hai.

Real-world example:

- Blog page me `article` content ke liye, `nav` navigation ke liye, `footer` bottom links ke liye.

Common mistakes:

- Har jagah sirf `div` use karna.
- Multiple `main` tags use karna.

Interview questions:

- What is semantic HTML?
- Why are semantic tags important?
- `section` vs `article`?

Mini task:

- Create a blog layout using semantic tags only.
