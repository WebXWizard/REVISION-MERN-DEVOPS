# Tables

Tables tabular data show karne ke liye use hote hain.

Tags:

- `table`
- `thead`
- `tbody`
- `tr`
- `th`
- `td`

Why used:

- Rows and columns wale data ke liye.

Real-world example:

- Student marks, invoice, orders list.

Common mistakes:

- Layout banane ke liye table use karna.
- Header cells ke liye `td` use karna instead of `th`.

Interview questions:

- When should you use tables?
- `th` vs `td`?

- Create an orders table with product, price, quantity, and total.
