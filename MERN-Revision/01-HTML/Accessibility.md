# Accessibility

Accessibility ka matlab hai website sab users ke liye usable ho, including screen reader and keyboard users.

Important:

- Labels for inputs
- Alt text for images
- Keyboard navigation
- Proper color contrast
- Semantic HTML
- ARIA only when needed

Real-world example:

- Login form screen reader se properly read hona chahiye.

Common mistakes:

- Clickable `div` instead of button.
- Missing focus state.
- Placeholder ko label ki tarah use karna.

Interview questions:

- What is accessibility?
- Why is label important?
- Button vs clickable div?

Mini task:

- Make a form usable with keyboard only.
