# HTML Study Notes

HTML means HyperText Markup Language. It gives structure to a web page.

Important topics:

- Semantic tags: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`
- Forms: `form`, `input`, `label`, `select`, `textarea`, `button`
- Input types: `text`, `email`, `password`, `number`, `date`, `file`, `checkbox`, `radio`
- Tables: `table`, `thead`, `tbody`, `tr`, `th`, `td`
- Media: `audio`, `video`, `source`
- Browser storage: `localStorage`, `sessionStorage`, cookies
- SEO basics: title, meta description, headings, alt text
- Accessibility: labels, alt text, keyboard navigation, ARIA only when needed

Real-world example:

- A login page uses semantic HTML, labels, required fields, and proper input types.
- A product page uses headings, image alt text, and meta tags for SEO.

Internal working:

- Browser parses HTML and creates DOM.
- CSS styles the DOM.
- JavaScript reads and updates the DOM.

Common mistakes:

- Using only `div` everywhere.
- Missing `label` for form inputs.
- Using tables for layout.
- Missing `alt` text on images.
- Storing sensitive data in `localStorage`.
