# Forms

Forms user input collect karne ke liye use hote hain.

Common tags:

- `form`
- `input`
- `label`
- `select`
- `textarea`
- `button`

Why used:

- Login, signup, search, feedback, checkout jaisi flows ke liye.

Real-world example:

- Login form with email, password, remember me, submit button.

Internal working:

- Browser form fields ko DOM me represent karta hai.
- Submit event trigger hota hai.
- Data server ko send ho sakta hai.

Common mistakes:

- `label` missing.
- `button` type specify nahi karna.
- Frontend validation ko final security samajhna.

Interview questions:

- Why use labels in forms?
- Difference between GET and POST form submission?

Mini task:

- Build accessible login form.
