# SEO Basics

SEO means Search Engine Optimization. HTML structure search engines ko page understand karne me help karta hai.

Important:

- `title`
- `meta description`
- Proper heading order
- Semantic tags
- Image `alt`
- Clean URLs

Real-world example:

- Product page ka title: `Buy Running Shoes Online`.

Common mistakes:

- Multiple random `h1` tags.
- Missing meta description.
- Images without alt text.

Interview questions:

- How does HTML help SEO?
- Why are headings important?

Mini task:

- Add SEO tags to a product page.
