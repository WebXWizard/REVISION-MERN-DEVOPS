# Input Types

Input types browser ko batate hain ki user se kis type ka data lena hai.

Common types:

- `text`
- `email`
- `password`
- `number`
- `date`
- `file`
- `checkbox`
- `radio`

Why used:

- Better validation, mobile keyboard, accessibility, and UX.

Real-world example:

- Email field me `type="email"` use karne se browser basic email validation karta hai.

Common mistakes:

- Phone number ke liye blindly `number` use karna.
- Password field ko `text` rakh dena.
- Required fields mark na karna.

Interview questions:

- Why use different input types?
- Checkbox vs radio?
