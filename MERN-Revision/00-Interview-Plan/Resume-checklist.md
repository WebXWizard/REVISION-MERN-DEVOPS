# Resume Checklist

Your resume should prove fresher readiness.

Must have:

- Clear title: MERN Stack Developer / Full Stack Developer
- Skills grouped properly
- 2 to 3 strong projects
- GitHub and live links
- Project bullet points with impact
- Technologies used
- Education and contact details

Avoid:

- Fake experience
- Too many random technologies
- Broken links
- Long paragraphs
- Spelling mistakes

Mini task:

- Rewrite each project in this format: built what, used which tech, solved what problem.
