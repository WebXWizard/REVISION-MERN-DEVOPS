# Interview Plan

Goal: Prepare like you have a MERN Stack fresher interview in a few days.

Main focus:

- Strong JavaScript fundamentals
- React component thinking
- API and backend flow
- MongoDB schema design
- Authentication flow
- Project explanation
- Basic DSA
- Clear HR communication

7-day fast plan:

```txt
Day 1: JavaScript basics, scope, hoisting, closures, event loop
Day 2: React components, hooks, forms, routing, Context/Redux basics
Day 3: Node.js, Express.js, REST APIs, middleware, error handling
Day 4: MongoDB, Mongoose, indexes, aggregation, schema design
Day 5: Auth, JWT, cookies, sessions, role-based access
Day 6: Projects, system design, resume-based questions
Day 7: Mock interview, DSA revision, HR answers
```

How to answer technical questions:

```txt
Sir/Ma'am, in simple words...
It is used when...
For example, in my project...
Internally...
One common mistake is...
```

Mini task:

- Record a 2-minute introduction.
- Explain your best MERN project in 3 minutes.
- Solve 3 JS output questions daily.
