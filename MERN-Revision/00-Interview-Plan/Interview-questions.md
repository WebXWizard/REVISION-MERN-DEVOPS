# Interview Plan Questions

Warm-up questions:

1. Tell me about yourself.
2. Why did you choose MERN Stack?
3. Explain your best project.
4. What was the biggest challenge in your project?
5. Why should we hire you as a fresher?

Technical screening questions:

1. What happens when JavaScript code runs?
2. Explain React props vs state.
3. How does Node.js handle multiple users?
4. What is middleware in Express?
5. What is indexing in MongoDB?
6. Explain JWT authentication flow.

Scenario questions:

1. Your API is slow. How will you debug?
2. User login works locally but fails in production. What will you check?
3. React component is re-rendering too much. How will you optimize?
4. MongoDB query is taking time. What can you do?

Mini task:

- Pick 5 questions and answer each in 60 seconds.
