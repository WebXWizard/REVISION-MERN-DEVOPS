# Answer Framework

Use this structure in every technical answer:

```txt
Definition -> Why used -> Real-world example -> Internal working -> Common mistake -> Conclusion
```

Example:

```txt
Closure means a function remembers variables from its outer scope.
It is used for data privacy and maintaining state.
In React, event handlers and hooks often use closure behavior.
Internally, JS keeps the outer lexical environment linked with the function.
A common mistake is expecting old state to update immediately.
```
