# DSA Interview Questions

Array questions:

1. Find maximum and minimum in array.
2. Reverse an array.
3. Remove duplicates.
4. Find second largest element.
5. Two sum.

String questions:

1. Reverse a string.
2. Check palindrome.
3. Count character frequency.
4. First non-repeating character.
5. Check anagram.

HashMap questions:

1. Count frequency of numbers.
2. Find duplicate elements.
3. Two sum using map.

Stack/queue questions:

1. Valid parentheses.
2. Implement stack using array.
3. Implement queue using array.

Recursion questions:

1. Factorial.
2. Fibonacci.
3. Sum of array recursively.

Interview follow-ups:

1. What is time complexity?
2. Can you optimize this?
3. What are edge cases?
4. Dry run your code.
