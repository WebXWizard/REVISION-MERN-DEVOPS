# Complexity

Complexity code ki performance measure karti hai.

Time complexity:

- Code input size ke saath kitna time leta hai.

Space complexity:

- Code kitni extra memory use karta hai.

Common Big O:

- O(1)
- O(n)
- O(n log n)
- O(n^2)

Common mistakes:

- Nested loop always O(n^2) assume karna without checking bounds.

- Find complexity of 5 simple loops.
