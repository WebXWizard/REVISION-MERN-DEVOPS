# DSA Study Notes

For MERN fresher roles, DSA basics are enough but should be clear.

Important topics:

- Arrays
- Strings
- HashMap / object frequency
- Stack
- Queue
- Linked List basics
- Recursion
- Two pointers
- Sliding window basics

How to solve:

```txt
1. Understand input/output.
2. Ask edge cases.
3. Start with brute force.
4. Optimize.
5. Explain time and space complexity.
6. Dry run with example.
```

Common mistakes:

- Jumping to code without understanding.
- Not handling empty input.
- Not explaining complexity.
- Not dry running.
- Getting stuck silently in interview.

Mini task:

- Solve 2 array, 2 string, and 1 hashmap question daily.
