# Branching

Branch separate line of development hoti hai.

Why used:

- Feature work alag rakhne ke liye.
- Main branch stable rakhne ke liye.

Commands:

- `git branch`
- `git checkout -b feature-name`
- `git merge`

Interview questions:

- What is branch?
- Why use feature branches?

Mini task:

- Create feature branch and merge it.
