# Environment Variables

Environment variables sensitive/config values store karte hain.

Examples:

- `MONGO_URI`
- `JWT_SECRET`
- `PORT`
- `CLIENT_URL`

Why used:

- Secrets code me hardcode na ho.
- Different config for local and production.

Common mistakes:

- `.env` GitHub par push karna.

Interview questions:

- What are environment variables?
- Why not commit `.env`?

Mini task:

- Create `.env.example` for your project.
