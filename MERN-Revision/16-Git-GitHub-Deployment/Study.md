# Git, GitHub, and Deployment Study Notes

Git tracks code changes. GitHub hosts repositories. Deployment makes your app available online.

Important topics:

- Repository
- Commit
- Branch
- Merge
- Pull request
- Remote
- `.gitignore`
- Environment variables
- Frontend deployment
- Backend deployment
- Database connection in production

Real-world MERN deployment:

```txt
Frontend: Vercel/Netlify
Backend: Render/Railway/Fly.io
Database: MongoDB Atlas
Environment variables: stored in hosting dashboard
```

Common mistakes:

- Committing `.env`.
- Not adding proper README.
- Broken live links.
- Hardcoding API URLs.
- Not configuring CORS for production domain.
