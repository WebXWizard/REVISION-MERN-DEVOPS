1. What is Git?
2. Git vs GitHub.
3. What is a commit?
4. What is a branch?
5. What is a pull request?
6. What is `.gitignore`?
7. Why should `.env` not be committed?
8. How do you deploy a MERN app?

1. App works locally but fails after deployment. What will you check?
2. Frontend cannot call backend in production. What could be wrong?
3. MongoDB connection fails in production. What will you check?
4. You accidentally committed `.env`. What should you do?
