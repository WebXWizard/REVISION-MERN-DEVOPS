# Frontend Deployment

Common platforms:

- Vercel
- Netlify

Steps:

```txt
Push code to GitHub -> import repo -> set build command -> set env variables -> deploy
```

Common mistakes:

- Wrong build command.
- Missing environment variables.
- API URL not configured.

Interview questions:

- How do you deploy React/Next app?
- What is build command?
