# Flexbox vs Grid

Flexbox one-dimensional hai. Grid two-dimensional hai.

Use Flexbox:

- Navbar
- Button groups
- Simple row/column alignment

Use Grid:

- Dashboard
- Image gallery
- Product listing

Interview answer:

```txt
Flexbox is best when layout is in one direction. Grid is better when we need control over rows and columns together.
```

Common mistake:

- Har layout ke liye same tool use karna.

Mini task:

- Build same card layout once with Flexbox and once with Grid.
