# Box Model

CSS box model har element ko box ke form me treat karta hai.

Parts:

- Content
- Padding
- Border
- Margin

Why used:

- Spacing and layout control ke liye.

Common mistake:

- Width me padding/border add hone ka behavior ignore karna.

Useful CSS:

```css
* {
  box-sizing: border-box;
}
```
