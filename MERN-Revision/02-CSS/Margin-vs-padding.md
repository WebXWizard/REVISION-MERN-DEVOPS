# Margin vs Padding

Margin element ke outside space ko control karta hai. Padding element ke inside space ko control karta hai.

Real-world example:

- Button ke andar text aur border ke beech space: padding
- Two cards ke beech space: margin or gap

Common mistakes:

- Component ke internal spacing ke liye margin use karna.
- Margin collapse behavior ignore karna.

Interview questions:

- Margin vs padding?
- What is margin collapse?

Mini task:

- Create two cards and use padding inside, margin outside.
