# Responsive Design

Responsive design ka matlab UI different screen sizes par properly work kare.

Why used:

- Mobile, tablet, desktop users ke liye same app usable rahe.

Techniques:

- Flexible layout
- Media queries
- Relative units
- Responsive images
- Mobile-first CSS

Common mistakes:

- Fixed widths everywhere.
- Text overflow check na karna.
- Mobile testing skip karna.

Interview questions:

- What is responsive design?
- Mobile-first vs desktop-first?

Mini task:

- Make a card layout responsive for mobile.
