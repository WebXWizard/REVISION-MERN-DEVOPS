# Grid

CSS Grid two-dimensional layout system hai.

Why used:

- Rows and columns dono control karne ke liye.

Important properties:

- `display: grid`
- `grid-template-columns`
- `grid-template-rows`
- `gap`
- `grid-column`
- `grid-row`

Real-world example:

- Dashboard, gallery, product grid.

Common mistakes:

- Simple one-line alignment ke liye Grid overuse karna.
- Responsive tracks define na karna.

Interview questions:

- What is CSS Grid?
- Flexbox vs Grid?
