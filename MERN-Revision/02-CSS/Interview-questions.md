# CSS Interview Questions

Concept questions:

1. What is CSS?
2. Explain the box model.
3. Margin vs padding.
4. Flexbox vs Grid.
5. `position: relative` vs `absolute`.
6. `display: none` vs `visibility: hidden`.
7. What is specificity?
8. Why does z-index sometimes not work?
9. What are media queries?
10. What is responsive design?

Coding questions:

1. Center a div horizontally and vertically.
2. Create a responsive navbar.
3. Create a 3-column grid layout.
4. Create a modal overlay.

Scenario questions:

1. UI breaks on mobile. What will you check?
2. A dropdown appears behind another element. How will you fix it?
3. Two CSS rules conflict. Which one applies and why?

Follow-up questions:

1. What is `box-sizing: border-box`?
2. Difference between `em`, `rem`, `%`, `px`.
3. How do you make images responsive?
