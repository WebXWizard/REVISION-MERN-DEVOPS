# Flexbox

Flexbox one-dimensional layout system hai.

Why used:

- Row ya column me items align karne ke liye.

Important properties:

- `display: flex`
- `justify-content`
- `align-items`
- `gap`
- `flex-direction`
- `flex-wrap`

Real-world example:

- Navbar, button groups, cards row.

Common mistakes:

- `justify-content` and `align-items` confuse karna.
- Parent par `display: flex` lagana bhool jana.

Interview questions:

- What is Flexbox?
- justify-content vs align-items?
