# Media Queries

Media queries screen size ya device conditions ke basis par CSS apply karte hain.

Example:

```css
@media (max-width: 768px) {
  .grid {
    grid-template-columns: 1fr;
  }
}
```

Why used:

- Responsive layouts ke liye.

Common mistakes:

- Too many random breakpoints.
- Breakpoints content ke according na choose karna.

Interview questions:

- What are media queries?
- How do you make layout mobile responsive?

Mini task:

- Add breakpoints for mobile, tablet, desktop.
