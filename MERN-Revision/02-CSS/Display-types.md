# Display Types

`display` define karta hai element page me kaise behave karega.

Common values:

- `block`
- `inline`
- `inline-block`
- `none`
- `flex`
- `grid`

Why used:

- Layout and visibility control.

Common mistakes:

- Inline element par width/height expect karna.
- `display: none` and `visibility: hidden` confuse karna.

Interview questions:

- block vs inline?
- display none vs visibility hidden?
