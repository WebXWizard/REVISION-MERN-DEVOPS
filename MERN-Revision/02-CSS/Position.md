# Position

`position` element ke placement behavior ko control karta hai.

Values:

- `static`
- `relative`
- `absolute`
- `fixed`
- `sticky`

Why used:

- Modal, tooltip, sticky header, badge positioning.

Common mistakes:

- `absolute` ka nearest positioned parent samajhna miss karna.
- Whole layout absolute se banana.

Interview questions:

- relative vs absolute?
- fixed vs sticky?
