# Specificity

Specificity decide karti hai conflicting CSS me kaunsi rule apply hogi.

Order:

```txt
inline style > id > class/attribute/pseudo-class > tag
```

Why used:

- CSS conflicts debug karne ke liye.

Common mistakes:

- `!important` overuse.
- Selectors unnecessarily complex banana.

Interview questions:

- What is CSS specificity?
- Which rule wins and why?

Mini task:

- Write 3 conflicting rules and calculate winner.
