# Process

`process` Node.js ka global object hai.

Use cases:

- Environment variables
- Command line arguments
- Exit process
- Current working directory

Examples:

```js
process.env.PORT;
process.argv;
```

Common mistakes:

- `.env` commit karna.

Interview questions:

- What is process object?
- What is process.env?

Mini task:
