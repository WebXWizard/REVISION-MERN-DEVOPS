# Node.js vs Express.js

Node.js runtime hai. Express.js Node.js framework hai.

Difference:

```txt
Node.js: JS outside browser run karta hai
Express.js: APIs and server routes easy banata hai
```

Real-world example:

- Node provides HTTP module.
- Express simplifies routing and middleware.
