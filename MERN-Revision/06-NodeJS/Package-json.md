# package.json

`package.json` project metadata and dependencies define karta hai.

Contains:

- scripts
- dependencies
- devDependencies
- version
- main/type

Why used:

- Project setup reproducible banane ke liye.

Common mistakes:

- Scripts samjhe bina run karna.
- Lock file ignore karna.

Interview questions:

- What is package.json?
- What is package-lock.json?
