# NPM

NPM Node Package Manager hai.

Why used:

- Packages install, scripts run, dependencies manage.

Common commands:

- `npm install`
- `npm uninstall`
- `npm run dev`
- `npm update`

Common mistakes:

- Dependency and devDependency confuse karna.
- Random packages install karna without need.

Interview questions:

- What is NPM?
- dependencies vs devDependencies?

Mini task:

- Install and use one utility package.
