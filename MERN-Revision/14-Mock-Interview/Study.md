# Mock Interview Mode

Rules for practice:

- One question at a time.
- Answer without notes.
- Get feedback.
- Improve the same answer.
- Difficulty increases gradually.

Scoring:

```txt
0-3: Concept unclear
4-5: Basic idea but weak explanation
6-7: Good fresher-level answer
8-9: Strong answer with example/internal working
10: Interview-ready with clarity and confidence
```

Answer framework:

```txt
Simple definition
Why used
Example
Internal working
Common mistake
```

Mini task:

- Start with JavaScript.
- Answer 10 questions in a row.
- Rewrite weak answers.
