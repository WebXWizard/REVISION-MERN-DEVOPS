# Virtual DOM

Virtual DOM actual DOM ka lightweight JavaScript representation hai.

Why used:

- Efficient UI updates ke liye.

Internal working:

```txt
State changes -> new virtual DOM -> diffing -> minimal real DOM update
```

Common mistakes:

- Virtual DOM ko browser DOM samajhna.
