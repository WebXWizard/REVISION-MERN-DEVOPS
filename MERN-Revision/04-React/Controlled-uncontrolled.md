# Controlled vs Uncontrolled

Controlled component me form value React state se control hoti hai.

Uncontrolled component me value DOM/ref se access hoti hai.

Use controlled:

- Validation
- Conditional UI
- Form submission control

Use uncontrolled:

- Simple forms
- File input

- Controlled vs uncontrolled component?
