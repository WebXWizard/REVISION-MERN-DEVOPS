# Components

Components reusable UI building blocks hote hain.

Types:

- Functional components
- Class components

Why used:

- UI ko small reusable parts me divide karne ke liye.

Real-world example:

- `Navbar`, `ProductCard`, `LoginForm`, `Modal`.

Common mistakes:

- One huge component banana.
- Reusable logic and UI mix kar dena.

Interview questions:

- What is a component?
- Functional vs class component?

Mini task:

- Create reusable `Button` and `Card` components.
