# Refresh Token

Refresh token new access token generate karne ke liye use hota hai.

Why used:

- User ko baar-baar login nahi karna padta.

Important:

- Long-lived but more protected.
- Usually httpOnly cookie me store hota hai.

Common mistakes:

- Refresh token rotation na samajhna.

Interview questions:

- Access token vs refresh token?
- What happens when access token expires?

Mini task:

- Design refresh token endpoint flow.
