# JWT Flow

JWT flow:

```txt
User login -> server validates -> JWT created -> client stores/sends token -> server verifies token -> protected data returned
```

Why used:

- Stateless authentication.

Common mistakes:

- Token expiry na rakhna.
- Secret expose karna.

- Explain JWT flow.
- What are JWT parts?
