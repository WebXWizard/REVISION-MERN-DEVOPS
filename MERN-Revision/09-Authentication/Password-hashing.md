# Password Hashing

Password hashing plain password ko one-way hash me convert karta hai.

Why used:

- Database leak hone par passwords directly expose na ho.

Common tool:

- bcrypt

Important:

- Hashing is not encryption.
- Hash cannot be reversed.

Common mistakes:

- Plain password store karna.
- Custom hashing algorithm banana.
