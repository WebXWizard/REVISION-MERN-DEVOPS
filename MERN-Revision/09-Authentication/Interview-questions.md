# Authentication Interview Questions

Concept questions:

1. Authentication vs authorization.
2. What is JWT?
3. JWT structure.
4. Access token vs refresh token.
5. Cookies vs localStorage for tokens.
6. What is httpOnly cookie?
7. What is role-based authorization?
8. Why hash passwords?
9. bcrypt hashing vs encryption.
10. What happens on logout?

Flow questions:

1. Explain complete login flow.
2. Explain protected route flow.
3. Explain refresh token flow.
4. Explain role-based access flow.

Scenario questions:

1. User token expired. What should happen?
2. Admin route is visible to normal user. What should backend do?
3. User logs out but can still access API. What will you check?
4. Token stolen from localStorage. What risk exists?
