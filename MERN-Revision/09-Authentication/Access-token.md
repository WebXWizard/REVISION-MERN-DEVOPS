# Access Token

Access token short-lived token hota hai protected APIs access karne ke liye.

Why short-lived:

- Token leak hone par risk kam.

Where used:

- Authorization header or httpOnly cookie.

Common mistakes:

- Very long expiry.
- Sensitive data payload me store karna.

Interview questions:

- What is access token?
- Why should it expire quickly?

Mini task:

- Explain access token verification flow.
