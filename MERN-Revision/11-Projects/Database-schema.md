# Database Schema

Project schema interviewer ko prove karta hai ki tum data modeling samajhte ho.

Explain:

- Collections/models
- Important fields
- Relationships
- Indexes
- Validation

Common mistakes:

- Own schema fields yaad na hona.
- Embed/reference reason na explain karna.

Interview questions:

- Explain your database schema.
- Why did you embed/reference this data?

Mini task:

- Write schema summary for your main collections.
