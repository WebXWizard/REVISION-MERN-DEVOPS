# Project Deployment

Deployment flow:

```txt
Frontend -> Vercel/Netlify
Backend -> Render/Railway
Database -> MongoDB Atlas
Env variables -> hosting dashboard
```

Common mistakes:

- API URL hardcoded.
- CORS production origin missing.
- `.env` committed.

Interview questions:

- How did you deploy your project?
- App works locally but not deployed. What will you check?
