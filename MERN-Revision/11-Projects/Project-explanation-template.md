# Project Explanation Template

Use this 3-minute structure:

```txt
1. Project name and problem
2. Main features
3. Tech stack
4. Architecture
5. Database schema
6. Auth/API flow
7. Biggest challenge
8. What I improved/learned
9. How I would scale
```

Sample opening:

```txt
I built this project to solve...
The frontend is built with React and backend APIs are built with Node.js and Express.
MongoDB stores data because the data is document-oriented and flexible.
```

Mini task:

- Fill this template for your best project.
