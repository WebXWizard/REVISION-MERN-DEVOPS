# API Flow

API flow explain karta hai request backend me kaise process hoti hai.

Example:

```txt
Frontend form submit -> API request -> Express route -> auth/validation middleware -> controller -> MongoDB -> response
```

Common mistakes:

- API route names ya methods yaad na hona.
- Error response explain na kar pana.

Interview questions:

- Explain one API from your project.
- How do you handle errors?

Mini task:

- Pick login API and explain complete flow.
