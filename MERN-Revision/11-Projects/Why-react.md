# Why React

Good fresher answer:

```txt
I used React because it helps build reusable components and state-driven UI. In my project, components like Navbar, ProductCard, Modal, and Forms are reused across pages.
```

Mention:

- Components
- State
- Hooks
- Routing
- Reusability

Avoid:

- "React is popular" only.

Interview questions:

- Why React in your project?
- Which components did you reuse?

Mini task:

- List 5 reusable components from your project.
