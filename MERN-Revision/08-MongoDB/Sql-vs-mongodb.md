# SQL vs MongoDB

SQL relational database hota hai. MongoDB document database hota hai.

Difference:

```txt
SQL: tables, rows, joins, fixed schema
MongoDB: collections, documents, flexible schema
```

Why MongoDB in MERN:

- JSON-like data naturally fit hota hai.
- Flexible schema.
- Good for rapid development.

Common mistakes:

- MongoDB ko always better bolna.
