# CRUD

CRUD means Create, Read, Update, Delete.

MongoDB methods:

- `insertOne`
- `find`
- `findOne`
- `updateOne`
- `deleteOne`

Mongoose methods:

- `create`
- `find`
- `findById`
- `findByIdAndUpdate`
- `findByIdAndDelete`
