# Documents

Document MongoDB ka data record hota hai, JSON-like BSON format me.

Example:

```json
{
  "name": "Amit",
  "email": "amit@example.com"
}
```

Why used:

- Flexible schema and nested data support.

Common mistakes:

- Very large nested arrays store karna.
