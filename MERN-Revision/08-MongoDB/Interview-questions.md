# MongoDB Interview Questions

Concept questions:

1. What is MongoDB?
2. SQL vs MongoDB.
3. What is a collection?
4. What is a document?
5. What is BSON?
6. What is indexing?
7. Why are indexes used?
8. What is aggregation pipeline?
9. What is Mongoose?
10. What is populate?
11. Embedding vs referencing.
12. How do you design schema in MongoDB?

Coding/query questions:

1. Find all users with age greater than 18.
2. Update one product price.
3. Delete inactive users.
4. Create an index on email.
5. Group orders by user and calculate total amount.

Scenario questions:

1. Query is slow. What will you check?
2. User has many orders. Embed or reference?
3. Need analytics from orders. Which MongoDB feature?
4. Duplicate email should not be allowed. What will you do?
