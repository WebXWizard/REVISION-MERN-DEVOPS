# Relationships

MongoDB relationships embedding ya referencing se handle hoti hain.

Embed when:

- Data small and frequently read together.

Reference when:

- Data large, shared, or independently updated.

Example:

- User and orders usually referenced.

Common mistakes:

- SQL style joins everywhere.

- Embed vs reference?
