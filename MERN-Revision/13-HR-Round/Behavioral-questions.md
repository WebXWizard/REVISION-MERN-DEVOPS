# Behavioral Questions

Common questions:

- Tell me about a time you solved a bug.
- Tell me about a time you learned something quickly.
- How do you handle feedback?
- What do you do when stuck?

Use STAR:

```txt
Situation -> Task -> Action -> Result
```

Common mistakes:

- No real example.
- Blaming others.

Mini task:

- Write one STAR answer for debugging a project issue.
