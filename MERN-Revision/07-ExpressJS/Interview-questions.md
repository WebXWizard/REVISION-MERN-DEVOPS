# Express.js Interview Questions

Concept questions:

1. What is Express.js?
2. Node.js vs Express.js.
3. What is middleware?
4. Explain middleware execution order.
5. What are `req`, `res`, and `next`?
6. What is centralized error handling?
7. What is REST API?
8. What are common HTTP methods?
9. What are common status codes?
10. Explain JWT flow in Express.

Coding questions:

1. Create a GET route.
2. Create a POST route with body validation.
3. Create auth middleware.
4. Create centralized error middleware.
5. Create CRUD routes for users.

Scenario questions:

1. Request is stuck and no response is sent. What can be wrong?
2. Protected route is accessible without login. What will you check?
3. Frontend cannot call backend due to CORS. How will you fix?
4. API returns 500 but no useful error. What improvement is needed?
