# Authorization

Authorization check karta hai user kya access kar sakta hai.

Example:

- Admin can delete product.
- User can view own profile.

Why used:

- Role-based access control ke liye.

Common mistakes:

- Sirf frontend button hide karna.
- Backend role check na lagana.

Interview questions:

- Authentication vs authorization?
- How do you protect admin routes?
