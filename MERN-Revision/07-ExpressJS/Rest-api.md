# REST API

REST API resources ko HTTP methods se expose karta hai.

Methods:

- GET
- POST
- PUT/PATCH
- DELETE

Status codes:

- 200
- 201
- 400
- 401
- 403
- 404
- 500

Common mistakes:

- Wrong status codes.
- Verb in URL like `/getUsers`.
