# Rapid Revision Resources

Before interview checklist:

- Resume updated
- GitHub projects visible
- Live links working
- Project README clear
- Basic JS questions revised
- React hooks revised
- Auth flow revised
- MongoDB schema revised
- HR intro ready

Last-minute practice:

- 10 JS output questions
- 5 React questions
- 5 backend questions
- 3 MongoDB questions
- 2 DSA questions
- 1 project explanation
- 1 self introduction

Confidence checklist:

- I can explain my project.
- I can write basic JS code.
- I can build basic React components.
- I can explain API and DB flow.
- I can communicate honestly.
