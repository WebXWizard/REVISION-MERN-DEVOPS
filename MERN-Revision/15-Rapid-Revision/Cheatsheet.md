# Cheatsheet

JavaScript:

```txt
var: function scoped
let/const: block scoped
Promise microtasks run before setTimeout callbacks
Closure: function + lexical environment
```

React:

```txt
Props: parent to child
State: internal changing data
useEffect: side effects
useRef: mutable value without re-render
```

Backend:

```txt
Middleware: req -> middleware -> route -> response
JWT: header.payload.signature
401: unauthenticated
403: unauthorized
```

MongoDB:

```txt
Collection = group of documents
Index = faster query
Aggregation = pipeline processing
```
