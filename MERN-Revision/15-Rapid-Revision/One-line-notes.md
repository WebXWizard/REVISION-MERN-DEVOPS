# One-Line Notes

HTML:

- Semantic HTML improves meaning, SEO, and accessibility.

CSS:

- Flexbox is for one-dimensional layout, Grid is for two-dimensional layout.

JavaScript:

- Closures happen when a function remembers outer scope.

React:

- State change triggers re-render.

Node.js:

- Node handles I/O using non-blocking event loop.

Express:

- Middleware runs between request and response.

MongoDB:

- Indexes improve read queries but add write overhead.

Auth:

- Authentication checks identity, authorization checks permission.
