# Last-Minute Notes

Before interview:

- Revise JS output questions.
- Revise your project architecture.
- Revise auth flow.
- Revise MongoDB schema.
- Keep intro ready.
- Keep GitHub/live links ready.

Do not:

- Learn completely new topic at last moment.
- Memorize without understanding.
- Overclaim.

During interview:

- Think aloud.
- Ask clarification.
- Use examples.
- Accept if you do not know and explain how you would learn.
