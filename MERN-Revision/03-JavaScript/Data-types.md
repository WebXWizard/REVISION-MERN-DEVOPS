# Data Types

JavaScript data types two categories me hote hain.

Primitive:

- string
- number
- boolean
- undefined
- null
- bigint
- symbol

Non-primitive:

- object
- array
- function

Internal working:

- Primitive values value by value compare hote hain.
- Objects reference by reference compare hote hain.

Common mistakes:

- `typeof null` returns `"object"`.
- Arrays ka `typeof` bhi `"object"` hota hai.
