# Scope

Scope decide karta hai variable kaha accessible hai.

Types:

- Global scope
- Function scope
- Block scope
- Lexical scope

Why used:

- Variable access control and predictable code.

Internal working:

- JS scope chain me variable search karta hai.

Common mistakes:

- Block scoped `let/const` ko block ke bahar access karna.
- Global variables overuse karna.

Interview questions:

- What is scope?
- Lexical scope kya hota hai?
