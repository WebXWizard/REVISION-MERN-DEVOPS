# Map Filter Reduce

Array methods data transformation ke liye use hote hain.

`map`:

- Har item transform karta hai and new array return karta hai.

`filter`:

- Condition pass karne wale items return karta hai.

`reduce`:

- Array ko single value me convert karta hai.

Common mistakes:

- `map` ko side effects ke liye use karna.
- `reduce` accumulator initial value skip karna.

Mini task:

- Products array se total price calculate karo using `reduce`.
