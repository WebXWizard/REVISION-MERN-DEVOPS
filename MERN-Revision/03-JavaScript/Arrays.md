# Arrays

Array ordered collection hoti hai.

Why used:

- Lists store karne ke liye: users, products, messages.

Important methods:

- `push`
- `pop`
- `map`
- `filter`
- `reduce`
- `find`
- `some`
- `every`

Common mistakes:

- Original array mutate kar dena.
- `map` me return bhool jana.
