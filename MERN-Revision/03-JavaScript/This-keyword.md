# This Keyword

`this` current execution context ke object ko refer karta hai.

Rules:

- Global context
- Object method
- Constructor function
- Explicit binding
- Arrow function lexical `this`

Common mistakes:

- Arrow function ko object method me use karna jab dynamic `this` chahiye.

Interview questions:

- Explain `this` keyword.
- Arrow function me `this` ka behavior?

Mini task:

- Write same method with normal and arrow function and compare output.
