# Hoisting

Hoisting me declarations memory phase me top par allocate ho jati hain.

Important:

- `var` hoisted with `undefined`.
- `let` and `const` hoisted but Temporal Dead Zone me hote hain.
- Function declarations fully hoisted hoti hain.

Common mistakes:

- `let/const` ko hoisted nahi samajhna.
- Function expression ko call before initialization karna.

Interview questions:

- What is hoisting?
- What is Temporal Dead Zone?
