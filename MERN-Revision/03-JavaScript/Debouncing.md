# Debouncing

Debouncing function call ko delay karta hai until user action stop ho jaye.

Why used:

- Search input
- Resize event
- Auto-save

Example use case:

- User typing stop kare 500ms ke baad API call.

Common mistakes:

- Har keypress par API call karna.
- Timer clear na karna.

Interview questions:

- What is debouncing?
- Debouncing vs throttling?
