# Call Apply Bind

`call`, `apply`, and `bind` `this` explicitly set karne ke methods hain.

Difference:

```txt
call: arguments comma separated
apply: arguments array me
bind: new function return karta hai
```

Example use:

- Method borrowing
- Function context fix karna

Common mistakes:

- `bind` ko immediately execute samajhna.

Interview questions:

- call vs apply vs bind?
- bind kya return karta hai?

Mini task:

- Use `call` to borrow method from another object.
