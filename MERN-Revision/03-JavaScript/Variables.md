# Variables

Variables data store karne ke liye use hote hain.

Types:

- `var`
- `let`
- `const`

Interview focus:

- `var` function scoped hota hai.
- `let` and `const` block scoped hote hain.
- `const` reassignment prevent karta hai, object mutation nahi.

Common mistakes:

- `const` object ko immutable samajhna.
- `var` hoisting behavior ignore karna.
