# Next.js Interview Questions

Concept questions:

1. What is Next.js?
2. Why use Next.js over plain React?
3. What is App Router?
4. What are Server Components?
5. What are Client Components?
6. When do you write `"use client"`?
7. SSR vs CSR vs SSG.
8. What are dynamic routes?
9. What is middleware?
10. How does Next.js help SEO?

Coding questions:

1. Create a dynamic route for `/products/[id]`.
2. Create a route handler for `GET /api/users`.
3. Create a layout shared by multiple pages.
4. Create a client form component.

Scenario questions:

1. Page needs SEO and fresh data. Which rendering strategy?
2. Page is mostly static but should update every few minutes. What will you use?
3. You need button click and state. Server or Client Component?
4. Need to check auth before route loads. What will you use?
