# API Routes

Next.js me route handlers backend endpoints create karne ke liye use hote hain.

Example:

```txt
app/api/users/route.js
```

Why used:

- Small backend logic, form submit, auth callbacks, webhooks.

Common mistakes:

- Large backend ko blindly Next.js route handlers me daal dena.

Interview questions:

- What are API routes in Next.js?
- Route handler vs Express route?
