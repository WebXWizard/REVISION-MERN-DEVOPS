# Routing

Next.js file-system based routing use karta hai.

Example:

```txt
app/about/page.js -> /about
app/products/page.js -> /products
```

Why used:

- Manual route configuration ki need kam hoti hai.

Common mistakes:

- Folder banaya but `page.js` missing.

Interview questions:

- How routing works in Next.js?
- Nested routes kaise bante hain?
