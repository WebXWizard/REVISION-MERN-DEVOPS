# SSG

SSG means Static Site Generation.

What:

- HTML build time par generate hota hai.

Why used:

- Fast pages, SEO, rarely changing content.

Real-world example:

- Blog, docs, marketing pages.

Common mistakes:

- Frequently changing data ke liye pure SSG use karna.
