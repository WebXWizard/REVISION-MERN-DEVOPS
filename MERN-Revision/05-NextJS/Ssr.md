# SSR

SSR means Server-Side Rendering.

What:

- HTML har request par server par generate hota hai.

Why used:

- SEO and fresh data ke liye.

Real-world example:

- News page, user-specific dashboard.

Common mistakes:

- Static content ke liye SSR overuse karna.
