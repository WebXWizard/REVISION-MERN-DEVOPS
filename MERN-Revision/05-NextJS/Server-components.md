# Server Components

Server Components server par render hote hain and client JS kam bhejte hain.

Why used:

- Better performance
- Direct server data access
- Less client bundle

Important:

- Hooks like `useState` server component me nahi use hote.

Common mistakes:

- Server component me browser APIs use karna.

Interview questions:

- What are Server Components?
- Benefits of Server Components?
