# Pagination

Pagination large data ko pages me divide karta hai.

Why used:

- Faster API response
- Better frontend performance
- Less database load

Types:

- Offset pagination
- Cursor pagination

Common mistakes:

- Full collection fetch karna.
- Page and limit validate na karna.
