# System Design Interview Questions

Concept questions:

1. How do you design a REST API?
2. What is pagination?
3. What is rate limiting?
4. What is caching?
5. What is horizontal scaling?
6. What is load balancing?
7. What is database indexing?
8. What is WebSocket?

Design questions:

1. Design a URL shortener.
2. Design a chat app.
3. Design a library management system.
4. Design an e-commerce backend.
5. Design a job portal backend.

Scenario questions:

1. URL shortener code collision happens. What will you do?
2. Chat messages should appear instantly. Which tech helps?
3. E-commerce product list is slow. How will you improve?
4. Too many login attempts. What protection will you add?
