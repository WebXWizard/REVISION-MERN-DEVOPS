# MERN Answer Framework

Use this format for interview answers:

```txt
What it is -> Why it is used -> Where I used it -> Code flow -> Common mistake -> Short conclusion
```

For project questions, use:

```txt
Problem -> Features -> Tech stack -> Frontend flow -> Backend API -> Database schema -> Auth -> Deployment -> Challenge -> Improvement
```

Example:

```txt
In my MERN project, React handles UI and routing, Express exposes REST APIs, MongoDB stores user and product data, and JWT with httpOnly cookies secures login. I deployed the frontend and backend separately, configured environment variables, and handled CORS for production.
```
