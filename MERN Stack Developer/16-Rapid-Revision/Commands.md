# Commands

Node:

```bash
npm init -y
npm install express mongoose dotenv
npm run dev
```

React:

```bash
npm create vite@latest
npm install
npm run dev
npm run build
```

Git:

```bash
git status
git add .
git commit -m "message"
git push origin main
```

MongoDB:

```bash
mongosh
show dbs
use appdb
db.users.find()
```
