# Cheatsheet

MERN:

```txt
MongoDB, Express.js, React, Node.js.
```

React:

```txt
Builds component-based frontend UI.
```

Express:

```txt
Builds backend APIs and middleware.
```

MongoDB:

```txt
Stores data as flexible JSON-like documents.
```

JWT:

```txt
Signed token used to represent authenticated user claims.
```
