# DevOps Cloud AWS Revision Vault

This vault follows your `ROADMAP.md` and is built for fresher/interview preparation.

Use it like this:

1. Open `Study.md` for overview.
2. Open topic files like `Permissions.md`, `EC2.md`, `Dockerfile.md`, or `Jenkinsfile.md` for focused revision.
3. Use `Resources.md` for official docs and practice direction.
4. Practice `Interview-questions.md` aloud.
5. Complete the mini task in each topic file.

```txt
What it is -> Why used -> Real-world example -> Internal working -> Common mistakes -> Interview answer
```
