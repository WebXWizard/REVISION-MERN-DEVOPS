# Process Management

Processes are running programs in Linux.

Useful commands:

```bash
ps aux
top
htop
kill
systemctl status nginx
```

Why used:

- Check app status.
- Debug high CPU/memory.
- Stop stuck processes.

Common mistakes:

- Killing wrong process.
- Not checking service logs.

Interview questions:

- How do you check running processes?
- How do you stop a process?

Mini task:

- Find process ID of a running service.
