# Linux Fundamentals Study

Linux is the most common OS for servers and cloud deployments.

Important topics:

- File system hierarchy
- Basic commands
- Permissions
- Users and groups
- Process management
- Package management
- Logs

Why used in DevOps:

- Most production servers run Linux.
- CI/CD agents and containers often use Linux.
- Troubleshooting requires shell confidence.

Mini task:

- SSH into a Linux machine and check disk, memory, processes, and logs.
