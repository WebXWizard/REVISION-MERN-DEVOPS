# Cheatsheet

Linux:

```txt
chmod: change permissions
chown: change owner
ps/top: process check
tail -f: live logs
```

AWS:

```txt
IAM: access control
EC2: virtual server
S3: object storage
VPC: private network
Lambda: serverless compute
CloudWatch: logs and metrics
```

Docker/K8s:

```txt
Image: package
Container: running image
Pod: smallest K8s unit
Deployment: manages replicas
Service: stable networking
```
