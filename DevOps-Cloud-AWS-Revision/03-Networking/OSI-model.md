# OSI Model

OSI model has 7 layers.

Layers:

```txt
Application
Presentation
Session
Transport
Network
Data Link
Physical
```

Why used:

- Networking troubleshooting ko structured banata hai.

Interview tip:

- For DevOps, focus on application, transport, network layers.

Mini task:

- Map HTTP, TCP, IP, and Ethernet to OSI layers.
