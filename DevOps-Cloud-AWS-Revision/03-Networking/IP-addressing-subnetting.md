# IP Addressing and Subnetting

IP address network me device identify karta hai.

Types:

- Public IP
- Private IP

Subnet:

- Network ko smaller parts me divide karta hai.

AWS example:

- VPC CIDR: `10.0.0.0/16`
- Public subnet: `10.0.1.0/24`
- Private subnet: `10.0.2.0/24`
