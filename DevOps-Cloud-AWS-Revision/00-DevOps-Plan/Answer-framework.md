# Answer Framework

Use this in interviews:

```txt
In simple words, ___ means ___.
It is used because ___.
In real projects, for example ___.
Internally, it works by ___.
A common mistake is ___.
```

Example:

```txt
Docker is a containerization platform. It packages app code with dependencies so the app runs consistently across environments. In real projects, we Dockerize Node.js APIs before deploying them. Internally, Docker uses images and containers. A common mistake is creating very large images.
```
