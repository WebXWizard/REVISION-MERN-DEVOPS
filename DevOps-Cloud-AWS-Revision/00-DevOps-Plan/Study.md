# DevOps Plan Study

DevOps is a culture and set of practices that connects development and operations.

Interview focus:

- Linux and networking basics
- Git workflows
- Cloud fundamentals with AWS
- Docker containers
- CI/CD pipelines
- Kubernetes orchestration
- Infrastructure as Code
- Monitoring and observability

7-day sprint:

```txt
Day 1: Linux, Bash, Networking, Security
Day 2: Git and Python automation
Day 3: AWS IAM, EC2, S3, VPC, Lambda, CloudWatch
Day 4: Docker and Nginx
Day 5: Jenkins CI/CD and artifacts
Day 6: Kubernetes
Day 7: Terraform, Ansible, Prometheus, Grafana
```

Mini task:

- Explain DevOps in 60 seconds using one project deployment example.
