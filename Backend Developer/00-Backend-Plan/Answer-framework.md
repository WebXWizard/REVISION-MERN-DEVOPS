# Backend Answer Framework

Use this format for backend interview answers:

```txt
Problem -> Endpoint/API -> Validation -> Business logic -> Database -> Security -> Error handling -> Performance -> Monitoring
```

For project questions:

```txt
Problem -> Architecture -> API routes -> Database schema -> Auth -> Background jobs -> Deployment -> Monitoring -> Challenge -> Improvement
```

Good backend answers include:

- Request lifecycle
- Data model
- Auth and authorization
- Error handling
- Performance
- Security
- Operational thinking
