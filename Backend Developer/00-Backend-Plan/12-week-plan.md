# 12 Week Backend Developer Plan

Week 1:

- Backend fundamentals, HTTP, request lifecycle, status codes, headers, cookies, and REST basics.

Week 2:

- Programming language core, async programming, error handling, packages, environment variables, and project structure.

Week 3:

- API design, validation, pagination, filtering, sorting, API documentation, and versioning.

Week 4:

- SQL basics, joins, indexes, transactions, migrations, normalization, and query optimization.

Week 5:

- NoSQL basics, MongoDB, schema design, relationships, indexing, aggregation, and tradeoffs.

Week 6:

- Authentication, authorization, JWT, sessions, OAuth basics, RBAC, password hashing, and secure cookies.

Week 7:

- Architecture, layered design, services, repositories, MVC, dependency injection, and clean code.

Week 8:

- Testing, debugging, logging, integration tests, API tests, linting, and formatting.

Week 9:

- Caching, Redis basics, queues, background jobs, async processing, and event-driven basics.

Week 10:

- Performance, scalability, rate limiting, load balancing, high availability, and database optimization.

Week 11:

- Security, DevOps basics, Docker, CI/CD, cloud deployment, monitoring, health checks, and incidents.

Week 12:

- Projects, system design, resume points, top questions, HR, and mock interviews.
