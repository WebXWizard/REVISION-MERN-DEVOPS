# Crossplane

Crossplane lets teams manage cloud infrastructure using Kubernetes-style APIs.

Use cases:

- Self-service cloud resources
- Platform-defined infrastructure APIs
- Composable infrastructure
- GitOps-based provisioning

Concepts:

- Provider
- Managed resource
- Composite resource
- Composition

Tradeoff:

```txt
Crossplane can improve self-service, but platform teams must design APIs carefully.
```
