# Troubleshooting

Debug order:

1. Confirm user or developer impact.
2. Check recent changes.
3. Check service health.
4. Check DNS and networking.
5. Check logs and metrics.
6. Check permissions and secrets.
7. Mitigate.
8. Document root cause.

Useful signals:

- Error rate
- Latency
- Pod restarts
- Deployment status
- Agent queue time
- Failed pipeline logs
