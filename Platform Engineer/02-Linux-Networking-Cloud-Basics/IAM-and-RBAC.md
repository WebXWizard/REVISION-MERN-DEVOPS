# IAM And RBAC

IAM controls access to cloud resources.

RBAC controls access based on assigned roles.

Platform concerns:

- Who can deploy?
- Who can provision infrastructure?
- Who can read secrets?
- Who can change policies?
- Which workloads need permissions?

Best practices:

- Use least privilege.
- Use roles or managed identities.
- Avoid long-term static credentials.
- Audit access regularly.
