# Incident Operations

Platform incidents can block many teams.

Examples:

- Cluster outage
- CI/CD outage
- GitOps controller failure
- Registry unavailable
- DNS or ingress issue
- Secret manager outage

Response:

1. Confirm developer impact.
2. Communicate status.
3. Mitigate.
4. Restore platform capability.
5. Write postmortem.
6. Improve guardrails or automation.
