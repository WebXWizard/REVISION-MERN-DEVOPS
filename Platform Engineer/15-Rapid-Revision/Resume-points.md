# Resume Points

Example Platform Engineer resume bullets:

- Designed golden path service templates with repository setup, Dockerfile, CI/CD pipeline, Kubernetes manifests, observability defaults, and documentation.
- Built internal developer platform concepts with service catalog, ownership metadata, templates, and developer onboarding flow.
- Implemented Kubernetes platform guardrails using namespaces, RBAC, resource quotas, and policy as code.
- Created reusable Terraform modules for repeatable cloud infrastructure provisioning.
- Practiced GitOps deployments using Argo CD or Flux with PR-based environment changes and rollback workflow.
- Added Prometheus and Grafana dashboards for cluster health, pod restarts, deployment status, and platform SLOs.
- Designed secure self-service workflows for environment creation, secrets access, and infrastructure requests.
- Defined platform success metrics including time to first deploy, golden path adoption, and developer satisfaction.
