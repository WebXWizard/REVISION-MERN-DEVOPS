# Cheatsheet

Platform Engineering:

```txt
Building internal platforms that help developers ship software safely and quickly.
```

IDP:

```txt
Internal Developer Platform with self-service workflows, templates, catalog, CI/CD, infra, and observability.
```

Golden path:

```txt
Recommended supported way to build and deploy a service.
```

Guardrails:

```txt
Safety controls that guide teams toward secure and reliable defaults.
```

Platform as product:

```txt
Treat internal developers as customers and improve the platform based on feedback and metrics.
```
