# Resume Points

Example DevOps Engineer resume bullets:

- Built CI/CD pipelines with test, build, artifact, Docker image, deployment, verification, and rollback stages.
- Dockerized applications using optimized Dockerfiles and pushed images to container registries with traceable tags.
- Deployed applications to Kubernetes using deployments, services, ingress, ConfigMaps, secrets, and rollout commands.
- Provisioned cloud infrastructure with Terraform using variables, outputs, modules, and remote state concepts.
- Automated server configuration using Ansible playbooks, roles, inventory, and idempotent tasks.
- Created Prometheus and Grafana dashboards for API latency, error rate, traffic, and infrastructure health.
