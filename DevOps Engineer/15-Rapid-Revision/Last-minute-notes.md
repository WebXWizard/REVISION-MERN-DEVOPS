# Last Minute Notes

Remember:

- Explain workflows, not only tools.
- Always mention monitoring and rollback.
- Use exact commands when possible.
- Secrets should not be hardcoded.
- Docker image tags should be traceable.
- Terraform state must be protected.
- Kubernetes debugging starts with describe, logs, and events.
- Good pipelines build once and deploy the same artifact.
- DevOps answers should include automation and reliability.
