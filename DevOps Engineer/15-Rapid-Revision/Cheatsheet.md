# Cheatsheet

DevOps:

```txt
Practices that improve collaboration, automation, delivery, and operations.
```

CI:

```txt
Automatically build and test code changes.
```

CD:

```txt
Automatically deliver or deploy validated changes.
```

IaC:

```txt
Manage infrastructure using code.
```

Observability:

```txt
Understand system behavior using metrics, logs, and traces.
```
