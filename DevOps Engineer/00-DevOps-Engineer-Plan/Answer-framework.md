# DevOps Answer Framework

Use this format for interview answers:

```txt
Manual problem -> Automation or pipeline -> Tools used -> Infrastructure -> Security -> Monitoring -> Rollback -> Improvement
```

Example:

```txt
Instead of manually deploying a Node.js app, I would create a CI/CD pipeline that runs tests, builds a Docker image, pushes it to a registry, deploys to Kubernetes or EC2, checks health, alerts on failure, and supports rollback to the previous version.
```

Good answers include:

- Automation reason
- Tools and workflow
- Security and secrets
- Monitoring
- Rollback
- Lessons learned
