# 12 Week DevOps Engineer Plan

Week 1:

- Linux, processes, services, logs, permissions, package management, and troubleshooting.

Week 2:

- Networking, DNS, HTTP, TLS, SSH, firewalls, security basics, and server debugging.

Week 3:

- Git, GitHub, branching, pull requests, merge conflicts, release collaboration, and Git workflows.

Week 4:

- Bash scripting, Python automation, cron jobs, health checks, and safe automation.

Week 5:

- CI/CD concepts, Jenkins, GitHub Actions, artifacts, secrets, and deployment strategies.

Week 6:

- Docker images, containers, Dockerfile, Compose, volumes, networks, and registries.

Week 7:

- Kubernetes architecture, pods, deployments, services, ingress, ConfigMaps, secrets, and debugging.

Week 8:

- Cloud fundamentals, IAM, compute, storage, networking, load balancing, and managed services.

Week 9:

- Terraform, providers, resources, variables, modules, state, plans, and cloud provisioning.

Week 10:

- Ansible, inventory, playbooks, roles, idempotency, and configuration management.

Week 11:

- Prometheus, Grafana, logs, alerts, dashboards, SLOs, incidents, runbooks, and postmortems.

Week 12:

- DevSecOps, projects, resume points, top questions, and mock interviews.
