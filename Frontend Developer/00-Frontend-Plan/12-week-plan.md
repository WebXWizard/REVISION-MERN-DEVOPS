# 12 Week Frontend Developer Plan

Week 1:

- HTML, semantic tags, forms, accessibility, browser storage, and SEO basics.

Week 2:

- CSS box model, specificity, Flexbox, Grid, responsive design, and animations.

Week 3:

- JavaScript variables, data types, functions, arrays, objects, scope, closures, and output questions.

Week 4:

- Async JavaScript, promises, async await, event loop, DOM, fetch, and error handling.

Week 5:

- TypeScript basics, interfaces, union types, generics, React props, and API response types.

Week 6:

- React components, props, state, hooks, forms, routing, and component design.

Week 7:

- Context API, Redux Toolkit, Zustand, React Query, server state, and performance basics.

Week 8:

- Next.js App Router, layouts, rendering strategies, server and client components, and deployment.

Week 9:

- UI design basics, accessibility patterns, responsive layouts, reusable components, and design systems.

Week 10:

- Testing, debugging, React Testing Library, Playwright basics, linting, and formatting.

Week 11:

- Performance, Core Web Vitals, SEO, image optimization, bundle optimization, and web security basics.

Week 12:

- Deployment, projects, resume points, top questions, and mock interviews.
