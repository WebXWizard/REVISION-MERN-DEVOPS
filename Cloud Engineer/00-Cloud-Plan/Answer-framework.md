# Cloud Answer Framework

Use this format for cloud interview answers:

```txt
Problem -> Requirement -> Cloud services -> Architecture -> Security -> Scaling -> Monitoring -> Cost -> Limitation
```

Example:

```txt
For a web app, I would use a VPC with public and private subnets. The load balancer stays public, application servers run in private subnets, and the database stays private. IAM roles provide permissions, CloudWatch monitors logs and metrics, Auto Scaling handles traffic, and backups protect data.
```

Good cloud answers include:

- Why a service is chosen
- Network placement
- IAM and security
- Monitoring and logs
- Backup and recovery
- Cost awareness
- Operational tradeoffs
