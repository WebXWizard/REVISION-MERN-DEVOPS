# Cloud Engineer Study Plan

A Cloud Engineer builds and operates infrastructure on cloud platforms.

Focus areas:

- Cloud service selection
- Networking and security
- Compute, storage, and databases
- Monitoring and operations
- Automation and Infrastructure as Code
- Cost control
- Disaster recovery
- Clear architecture explanation

Suggested order:

1. Cloud fundamentals
2. Linux, networking, and security
3. AWS IAM and core services
4. Compute, storage, and database
5. VPC, DNS, load balancing, and CDN
6. Monitoring and operations
7. IaC and automation
8. Containers and Kubernetes
9. High availability and disaster recovery
10. Cost optimization and governance
11. Multi-cloud basics
12. Projects and mock interviews
