# Migration Strategies

Common cloud migration strategies:

- Rehost: lift and shift.
- Replatform: small cloud optimizations.
- Refactor: redesign for cloud-native.
- Repurchase: move to SaaS.
- Retire: remove unused apps.
- Retain: keep as-is.

Migration concerns:

- Downtime
- Data transfer
- Security
- Cost
- Dependencies
- Rollback plan
