# IaaS, PaaS, And SaaS

IaaS:

- Infrastructure as a Service.
- Example: EC2 virtual machines.
- You manage OS, app, and runtime.

PaaS:

- Platform as a Service.
- Example: Elastic Beanstalk or App Service.
- Provider manages more infrastructure.

SaaS:

- Software as a Service.
- Example: Gmail, Salesforce, GitHub.
- User consumes finished software.

Interview line:

```txt
IaaS gives infrastructure, PaaS gives a managed platform, and SaaS gives ready-to-use software.
```
