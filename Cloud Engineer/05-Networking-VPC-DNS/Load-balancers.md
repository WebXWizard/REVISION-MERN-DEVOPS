# Load Balancers

Load balancers distribute traffic across targets.

Common types:

- Application Load Balancer for HTTP and HTTPS.
- Network Load Balancer for TCP and high performance.

Benefits:

- High availability
- Health checks
- Traffic distribution
- TLS termination
- Path-based routing

Watch:

- Target health
- 4xx and 5xx errors
- Latency
- Request count
