# Multi-cloud Strategy

Reasons for multi-cloud:

- Avoid vendor lock-in
- Meet customer or compliance needs
- Improve resilience
- Use best service from each provider

Risks:

- More complexity
- More networking challenges
- More IAM models
- Higher skill requirement
- Harder cost tracking

Good approach:

```txt
Use multi-cloud only when the business reason is stronger than the operational complexity.
```
