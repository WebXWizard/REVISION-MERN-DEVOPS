# AWS, Azure, And Google Cloud Map

Service mapping:

```txt
AWS EC2 -> Azure Virtual Machines -> Google Compute Engine
AWS S3 -> Azure Blob Storage -> Google Cloud Storage
AWS IAM -> Azure Entra ID/RBAC -> Google Cloud IAM
AWS VPC -> Azure Virtual Network -> Google VPC
AWS Lambda -> Azure Functions -> Google Cloud Functions
AWS RDS -> Azure SQL/Database services -> Cloud SQL
AWS CloudWatch -> Azure Monitor -> Google Cloud Monitoring
```

Remember:

```txt
Names differ, but core cloud concepts are similar.
```
