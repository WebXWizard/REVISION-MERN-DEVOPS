# Cheatsheet

Cloud computing:

```txt
Using compute, storage, networking, and managed services over the internet.
```

IaaS:

```txt
Infrastructure like virtual machines and networks.
```

PaaS:

```txt
Managed platform for running applications.
```

SaaS:

```txt
Ready-to-use software.
```

Shared responsibility:

```txt
Provider secures the cloud; customer secures what they build in the cloud.
```

High availability:

```txt
Designing systems to remain usable during failures.
```
