# Last Minute Notes

Remember:

- Always include security in cloud answers.
- Put databases in private subnets.
- Use IAM roles for applications.
- Enable MFA for human users.
- Monitor with metrics, logs, alarms, and dashboards.
- Backups must be restore-tested.
- Use multiple AZs for high availability.
- Use tags for ownership and cost tracking.
- Use IaC for repeatable infrastructure.
- Explain tradeoffs between cost, reliability, performance, and security.
