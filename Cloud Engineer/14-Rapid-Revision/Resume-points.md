# Resume Points

Example Cloud Engineer resume bullets:

- Designed secure AWS VPC architecture with public and private subnets, load balancer, app servers, and private database.
- Deployed static website using object storage, CDN, custom domain, and TLS.
- Built serverless API using API Gateway, Lambda, DynamoDB, IAM roles, and CloudWatch logs.
- Provisioned cloud infrastructure using Terraform with variables, outputs, and reusable configuration.
- Created CloudWatch dashboards and alarms for compute, application, and load balancer metrics.
- Implemented tagging and budget alerts to improve cloud cost visibility.
- Practiced backup, restore, and disaster recovery planning using RTO and RPO.
