# Commands

Linux:

```bash
ssh -i key.pem user@server-ip
df -h
free -h
top
systemctl status nginx
journalctl -u nginx
```

Networking:

```bash
curl -I https://example.com
dig example.com
ping example.com
ss -tulpn
```

AWS CLI:

```bash
aws sts get-caller-identity
aws s3 ls
aws ec2 describe-instances
aws cloudwatch describe-alarms
```

Terraform:

```bash
terraform init
terraform plan
terraform apply
terraform destroy
```
